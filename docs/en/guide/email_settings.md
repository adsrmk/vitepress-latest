# Email Client Settings

Technical credentials required to manually connect your email account to third-party applications like Outlook, Apple Mail, or Gmail.


### Connection Protocols


IMAP: Use server mail.evworkspace.nl on port 993 with SSL/TLS for modern multi-device syncing.
SMTP: Connect for outgoing mail using server mail.evworkspace.nl on port 465 (SSL/TLS) or 587 (STARTTLS).
POP: Access server mail.evworkspace.nl on port 995 with SSL/TLS for simple message downloading.
Username: Always use your full email address (e.g., admin@evworkspace.nl) as the login credential.
