# Monitor real-time server data  <Badge type="tip" text="Enterprise" />

As an **Enterprise client**, you have access to server analytics and monitoring.
This provides your team with 1:1 transparency into the health and resource consumption of your managed server(s).
Rather than waiting for a monthly report, we use **[beszelâ†—](https://beszel.dev/guide/what-is-beszel)**, a lightweight web-app that gives you a real-time insights into your environment, helping you make data-driven decisions about scaling and optimization.

Upon registering as an customer, you'll get an email with login details to view certain server analytics such as:

- Memory Resident Set (RSS): Track the actual RAM footprint of your apps.
- CPU & Load: Identify peak traffic hours.
- Network: Monitor bandwidth spikes during data transfers.
- Disk Usage and I/O: How many disk usage has been utilized and when read/writes occur.
