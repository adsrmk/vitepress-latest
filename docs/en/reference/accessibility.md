# Accessibility Checker

[AccessibilityChecker](https://AccessibilityChecker.org/) is a web accessibility testing and compliance platform. 
It lets you scan your website for issues that could make it difficult or impossible for people with disabilities to use your site. 
The tool checks against major accessibility standards like **WCAG**, **ADA**, **Section 508**, **AODA**, and the **European Accessibility Act**.  ￼
